# Politica obligatoria de imagenes - TechZen Smart Homes

Esta web no debe usar imagenes copiadas de webs de electricistas, instaladores, empresas de domotica, KNX partners, Shelly partners, Loxone partners ni competidores.

## Fuentes permitidas

- Fotos propias de trabajos reales de TechZen Smart Homes.
- Imagenes generadas especificamente para TechZen Smart Homes.
- Mockups propios creados para la web.
- Bancos gratuitos o recursos libres con licencia comercial clara, por ejemplo Unsplash, Pexels, Pixabay o Wikimedia Commons cuando la licencia lo permita.
- SVGs y placeholders creados dentro del proyecto.

## Fuentes no permitidas

- Webs de competidores.
- Webs de instaladores, electricistas, integradores, KNX partners, Shelly partners o Loxone partners.
- Imagenes sin licencia clara.
- Imagenes con marcas de agua.
- Imagenes con logos de terceros no autorizados.
- Imagenes con nombres de otras empresas, textos comerciales ajenos, direcciones, matriculas o caras reconocibles.

## Regla de decision

Si hay duda sobre la licencia, el origen, la autorizacion comercial o la privacidad de la imagen, no se usa.

## Como sustituir fotos

Las imagenes estan en:

```text
public/images/
```

Para sustituir una imagen, usa una foto propia o una imagen con licencia clara y manten el mismo nombre de archivo cuando sea posible. Despues revisa el texto `alt` correspondiente en `src/App.jsx`.
